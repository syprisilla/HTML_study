transition

구조 transition:속성명 지속시간 타이밍함수 딜레이;

속성명:어떤 속성이 변화할지 지정(ex:background-color, margin-left)

지속시간:애니메이션이 얼마나 걸릴지(ex:1s, 500ms)

타이밍 함수:변화 속도의 곡선을 결정함(ease, linear 등)

딜레이: 얼마나 기다렸다가 시작할지

transition-timing-function(변화 속도 곡선)

        요소가 어떤 속도 곡선으로 움직일지 결정함

        기본 제공 함수
        linear 일정한 속도
        ease 천천히 시작->빨라짐->천천히 끝남
        ease-in 천천히 시작->점점 빨라짐
        ease-out 빠르게 시작->천천히 멈춤
        ease-in-out 천천히 시작->빠르게->다시 처천히

        cubic-bezier(커스텀 곡선)
        베지어 곡선을 정의해 자유로운 속도 변화를 만들 수 있음
        ex:transition-timing-function: cubic-bezier(0.65, 0, 0.35, 1)

transform

요소를 원래 위치에서 벗어나지 않고 바꾸는 CSS속성

종류

        translate(x,y)
        이동시키는 함수
        ex:translateX(200px) 오른쪽으로 200px
           translate(-200px,50px) 왼쪽 200px, 아래 50px

        scale(x,y)
        크기 변경
        ex:scale(2) 2배 확대
           scale(0.5) 절반으로 축소
           scaleY(1.5) 높이만 1.5배 키움

        rotate
        회전
        ex:rotate(45deg) 45도 시계 방향 회전
           rotate(-0.6turn) 반시계 방향 0.6바퀴 회전
        (1turn=360도)  
        
        skew
        기울임
        ex:skew(10deg,20deg) x축으로 10도, y축으로 20도 기울임
           skew(300deg) 너무 많이 기울여서 깨져 보일 수 있음

        transform-origin
        회전이나 크기 변경의 기준점을 지정
        (기본값은 center center)

        transform 여러 개를 동시에 쓸 때 순서
        ex:transform: translateX(-500px) rotate(180deg) scaleY(1.5);
        해석:왼쪽으로 500px 이동-그 상태에서 180도 회전-거기서 세로만 1.5배 확대

        tip 세션 자체에 영향을 주면 자식 요소 전체에 영향을 줌
        ex:.transform section:nth-of-type(2)
            {
              transform: scale(0.7);
            }
