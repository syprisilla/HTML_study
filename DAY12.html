Media Queires

정의

웹 페이지를 다양한 화면 크기와 기기 환경에 맞게 스타일을 조정할 수 있도록 도와주는 CSS 기능

사용하는 이유

  • 다양한 기기 지원:기기에 따라 글자 크기, 배치, 색상 등을 변경해야 함
  • 가독성 향상:font-size조절
  • UX/UI 개선:버튼 등을 수직 정렬로 바꾸거나 숨김

기본 구조

        @media (조건문){
            조건을 만족할 때 적용할 CSS
        }

주요 조건 종류

  • min-width:해당 이상일 때 적용
  • max-width:해당 이하일 때 적용
  • min-height:높이 기준
  • max-height:높이 기준
  • orientation:화면 방향
  • aspect-ratio:가로/세로 비율
  • hover,pointer:입력방식

주의할 점

조건이 겹칠 경우 가장 마지막 media query가 적용됨

CSS Reset

정의

브라우저마다 기본 스타일이 다르기 때문에 이 차이를 없애기 위해 초기화 하는 CSS 코드

필요한 이유

  • 크로스 브라우징 문제 해결
  • 통일된 스타일 기반에서 디자인 시작 가능
        기본적인 reset 구조
        *{              모든 요소 선택
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        추가적으로 초기화하는 경우
        body {
           font-family: sans-serif; 기본 글꼴
           line-height: 1.5;
        }

       ul, ol {
         list-style: none;  점 없애기 
       }

       a {
          text-decoration: none;
          color: inherit;
        }
