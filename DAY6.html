박스 모델이란?

웹에서 우리가 보는 모든 요소(글자,사진,버튼 등)는 네모난 박스로 되어있음

Content 영역

        진짜 냐용이 들어가는 부분(글자,이미지 등)

Padding 영역

        내용과 테두리 사이의 공간
        안에 있는 내용이 너무 붙지 않게 여유를 주는 공간

Border 영역

        padding 바깥에 있는 선(테두리)
        색깔,두께,선의 종류를 정할 수 있음

Margin 영역

        다른 박스와 얼마나 떨어질지를 정하는 공간(바깥 여백)
        이 공간은 투명해서 색이나 이미지가 들어가지 않음

속성들

content

  • weight
  • height

padding

  • padding-top
  • padding-bottom
  • padding-left
  • padding-right

    • border-right

    • border-color

    • border-top-style

    • border-right-width
    • border-right-style
    • border-right-color
    • border-bottom-width
    • border-bottom-style
    • border-bottom-color
    • border-left-width
    • border-left-style
    • border-left-color

margin

  • margin-top
  • margin-bottom
  • margin-left
  • margin-right

속기

        padding,margin:top->right->bottom->left 시계방향
        border:굵기->스타일->색상
        ex border:1px dotted yellow

블록과 인라인

CSS는 크게 블록 박스와 인라인 박스 유형으로 구분

블록으로 정의된 경우

  • 인라인 방향으로 연장되어 상위 콘테이너에서 사용가능 공간 할당
  • 새 줄로 행갈이
  • width height 속성 존중됨
  • padding margin border로 인해 다른 요소들이 밀려남

표준 박스 모델

표준박스 모델 크기
        .box {
                      width: 350px;
                      height: 150px;
                      margin: 25px;
                      padding: 25px;
                      border: 5px solid black;
                    }
구성
        width,height:내용 크기
        padding,border:추가로 크기에 더해짐
        maring:바깥 여백, 별도 공간

        표준 박스 모델을 사용하여 박스가 차지하는 공간은 너비410px(350+25+25+5+5), 높이 210px(150+25+25+5+5)

대체 박스 모델

표준박스 모델 크기
        .box {
                      box-sizing: border-box;
                      width: 350px;
                      height: 150px;
                      margin: 25px;
                      padding: 25px;
                      border: 5px solid black;
                    }
구성
      width,height:내용+padding+border 포함된 전체 박스 크기
      margin:바깥 여백, 별도 공간
