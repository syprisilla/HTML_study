Bootstrap


정의

트위터에서 개발한 오픈소스 CSS 프레임 워크

반응형 웹 페이지를 빠르게 만들 수 있게 도와줌


특징

  • 미리 정의된 디자인 시스템
  • 반응형 레이아웃
  • 모바일 우선
  • 쉽고 빠른 개발
  • 다양한 유틸리티 클래스(margin, padding)

기본 문법 구조

        <head>
           필수 CSS 링크
           <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/css/bootstrap.min.css" rel="stylesheet">
        </head>

        <body>
          선택 JS (알림, 모달 등 동작용)
          <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/js/bootstrap.bundle.min.js"></script>
        </body>

        bootstrap.min.css:디자인을 위한 모든 클래스가 정의되어 있음
        bootstrap.bundle.min.js:JavaScript 기반의 동작 요소에 필요

컨테이너와 레이아웃 시스템

        <div class="container">...</div> : 고정 너비
        <div class="container-fluid">...</div> : 전체 화면 너비
        <div class="container-xl">...</div> : 화면 너비에 따라 다르게 적용됨(반응형)

유틸리티 클래스

  • 색상:bg-damger, text-primary, btn-warning
  • 정렬:text-center, float-end, d-flex
  • 여백:m-3, p-2, mb-5(m=margin, p=padding, t=top, b=bottom, s=start, e=end)
  • 글자 크기:fs-1, fs-2(fs=font-size)
  • 폰트 굵기:fw-bold, fw-light

버튼

주요 종류

  • btn-primary(파란색), btn-secondary(회색), btn-success(초록색), btn-danger(빨간색) 등
  • btn-outline-*:테두리만 있는 버튼
  • btn-lg, btn-sm:크기 조절
  • disabled:비활성화 상태
        기본
        <button type="button" class="btn btn-primary">Primary</button>
        테두리
        <button type="button" class="btn btn-outline-primary">Primary</button>
        비활성화
        <button type="button" class="btn btn-primary" disabled>Primary button</button>ㅋ

Badge

작은 숫자, 텍스트, 상태 드응ㄹ 강조해서 보여주는 UI요소

    기본 문법
    <span class="badge bg-primary">NEW</span>
    badge:배지 스타일 적용
    bg-primary:배경색 설정(파란색)
    글자는 작고 둥근 사각형 형태로 나타남

    position-relative/absolute
    부모 .position-relative 기준으로
    자식 .position-absolute가 상단 우측(top-0 start-100)으로 위치
    translate-middle:가운데 정렬 미세 조정
